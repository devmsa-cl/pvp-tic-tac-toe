# PVP Tic Tac Toe

A simple real-time player-vs-player (PVP) Tic Tac Toe game built with Socket.I0, Enabling two users to complete in a classic game with live update and highlighted score point.

## Feature

- Two Player can play against each other in real-time.
- Dynamic game board with visual highlight for moves.
- Live scoreboard tracking wins, losses.
- Uses the latest Socket.IO library for seamless communication.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
